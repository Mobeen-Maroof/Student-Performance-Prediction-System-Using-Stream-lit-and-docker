# Student Performance Prediction System Using Streamlit and Docker

## Developer By
1. Mobeen Maroof

## Project Description
The Student Performance Prediction System is a Streamlit-based Machine Learning web application that predicts whether a student is likely to pass or fail based on academic factors such as study hours, attendance, previous grades, and other relevant information. The application is containerized using Docker, ensuring easy deployment, portability, and consistent performance across different environments. The system helps educators identify at-risk students and make data-driven decisions to improve academic outcomes.

## Features
- Predict student performance (Pass/Fail)
- Interactive Streamlit dashboard
- Real-time prediction results
- Machine Learning model integration
- Student data input and analysis
- Data visualization and performance insights
- Docker containerization for easy deployment
- Cross-platform compatibility
- Accurate and efficient performance prediction

## Technologies Used
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- Plotly
- Docker
